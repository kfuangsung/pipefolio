class pipefolio.selector.AnnualVolatilitySelector(n, mode=SelectMode.SMALLEST)#

Bases: pipefolio.mixins.SelectorPipeMixin, sklearn.base.TransformerMixin, sklearn.base.BaseEstimator

Selector to choose assets based on their annualized volatility.

This selector computes the annualized volatility of asset returns and selects the top or bottom n assets based on the specified selection mode.

  • mode (pipefolio.enums.SelectMode)
